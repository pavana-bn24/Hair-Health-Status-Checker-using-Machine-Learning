# Hair Health Status Checker

## Project Description
Hair Health Status Checker is a machine learning based web application that evaluates hair health using lifestyle factors such as sleep hours, nutrition level, and stress level. The system analyzes user input and predicts whether the hair health risk is low, moderate, or high.

## Features
- Web-based user interface
- Machine learning prediction
- Flask API backend
- Hair health risk classification

## Technologies Used
- Python
- Flask
- Scikit-learn
- NumPy
- HTML & CSS

## How It Works
1. The user enters hair-related lifestyle factors such as sleep hours, nutrition level, and stress level.
2. The input data is sent to the Flask backend.
3. The machine learning model processes the input values.
4. The system predicts the hair health status.
5. The result is displayed on the webpage.

## Output
The system displays hair health status as:
- Strong Hair (Low Risk)
- Moderate Hair Fall Risk
- High Hair Fall Risk
